# Blinkit Grocery Sales Analysis 🛒

## 📊 Project Overview
End-to-end data analysis project on Blinkit grocery 
sales data using MySQL and Power BI.

## 🖼️ Dashboard Preview
![Blinkit Dashboard](dashboard_screenshot.png)

## 📁 Dataset
- Source: Kaggle
- Records: 8,523 rows
- Fields: Item Type, Fat Content, Outlet Size, 
  Location Type, Sales, Rating

## 🛠️ Tools Used
- **MySQL** — Data cleaning and analysis
- **Power BI** — Interactive dashboard building

## 🔍 Key Insights
- Fruits and Snack Foods are top selling categories
- Tier 3 cities generate highest revenue ₹472K
- Regular fat products outsell Low Fat 65% vs 35%
- Supermarket Type 1 dominates with ₹7.87M sales

## 📝 SQL Analysis Performed
- KPI metrics (Total Sales, Avg Sales, Rating)
- Product performance by category
- Outlet size and location analysis
- Year trend analysis
- Customer rating analysis
